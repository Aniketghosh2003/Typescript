## Features of TypeScript Practiced

- **Static Typing**: Adding type annotations to variables and functions.
- **Interfaces**: Defining the structure of objects.
- **Classes and Objects**: Utilizing object-oriented programming concepts.
- **Enums**: Defining a set of named constants.
- **Generics**: Writing reusable and flexible code.
- **Modules**: Organizing code using imports and exports.
